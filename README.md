TCG Card Game
Projeto de jogo de cartas desenvolvido para praticar lógica de programação, manipulação de estados, organização de componentes e regras de gameplay.

O jogo foi inspirado em TCGs clássicos, permitindo batalhas entre cartas com atributos, sistema de turno e gerenciamento de ações durante a partida.

Demonstração
Acesse o projeto: https://filipeizaak.github.io/Jogo-de-cartas/

Funcionalidades
Sistema de batalha entre cartas
Controle de turnos
Sistema de atributos
Interação dinâmica entre jogadores
Interface responsiva
Atualização de estado em tempo real
Estrutura modular para expansão de cartas e mecânicas
Tecnologias utilizadas
JavaScript
HTML5
CSS3
Objetivos do projeto
Este projeto foi criado com o objetivo de desenvolver habilidades práticas em:

lógica de programação
manipulação do DOM
gerenciamento de estado
arquitetura de componentes
organização de código
desenvolvimento web
Além disso, o projeto serviu como prática para estruturar regras complexas de gameplay de maneira escalável.
